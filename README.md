# Diabetes-Prediction-Model

Diabetes Prediction using Decision Tree Regressor
Overview

This project builds a machine learning model using Decision Tree Regressor to predict diabetes progression based on medical attributes. It demonstrates the complete ML pipeline including data preprocessing, model training, and evaluation.

Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

Workflow

Data loading and preprocessing

Exploratory Data Analysis (EDA)

Train-test split

Model training using Decision Tree Regressor

Model evaluation using MAE and R² score

Result

The model learns non-linear relationships between medical features and diabetes progression and provides predictions based on patient health parameters.
